# Tipos de Página — Estruturas Padrão

## Página de Vendas (`/nova-lp vendas`)

**Objetivo:** converter visitante em comprador direto

| Seção | Conteúdo |
|---|---|
| Hero | Headline impactante + subheadline + CTA + visual |
| Problema | Agitação das dores do público (3-5 bullets) |
| Solução | Como o produto/serviço resolve |
| Benefícios | O que o cliente ganha (não features — benefícios) |
| Prova Social | Depoimentos + números + logos |
| Oferta | O que está incluso + bônus + garantia |
| CTA intermediário | Repetição do botão principal |
| FAQ | 5-7 objeções + respostas |
| CTA Final + Urgência | Botão + elemento de escassez/urgência (se real) |

---

## Página de Captura (`/nova-lp captura`)

**Objetivo:** capturar lead (email / WhatsApp)

| Seção | Conteúdo |
|---|---|
| Hero + Formulário | Proposta de valor clara + form simples (nome + email/WhatsApp) |
| Benefícios rápidos | 3-5 bullets do que o lead vai receber |
| Prova social rápida | 1-2 depoimentos ou número de leads |
| CTA reforçado | Repetição do form + urgência |

---

## Página Institucional (`/nova-lp institucional`)

**Objetivo:** apresentar a empresa/profissional e gerar contato

| Seção | Conteúdo |
|---|---|
| Hero | Proposta de valor + CTA de contato |
| Sobre | História, missão, diferenciais |
| Serviços | Grid de serviços com ícones |
| Cases/Portfolio | Resultados de clientes |
| Depoimentos | Carrossel de feedbacks |
| Contato | Formulário + localização + WhatsApp |
