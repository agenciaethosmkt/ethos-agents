# Skills por Formato — Guia de uso do Ogilvy

## Mapeamento formato → skill

| Formato | Skill principal | Skill complementar |
|---|---|---|
| Post de Feed | `16-instagram-copy` | `19-instagram-hashtag` |
| Carrossel | `16-instagram-copy` | `19-instagram-hashtag` |
| Stories | `16-instagram-copy` | — |
| Reels | `18-instagram-roteiro` | `22-conteudo-script-video` |
| Vídeo (YouTube/outros) | `22-conteudo-script-video` | `18-instagram-roteiro` |
| E-mail Marketing | — (sem skill dedicada) | `16-instagram-copy` como base |

## Quando combinar skills

- **Reels com alta produção**: usar `18-instagram-roteiro` (estrutura narrativa) + `22-conteudo-script-video` (script técnico com cenas)
- **Feed educativo**: usar `16-instagram-copy` para legenda + `19-instagram-hashtag` para alcance
- **Carrossel de vendas**: usar `16-instagram-copy` com foco em gatilhos por slide

## Instruções de leitura de skills

As skills ficam em `~/.claude/skills/`. Antes de escrever a copy, ler a skill correspondente para aplicar os frameworks corretos.

Exemplo para Reels:
```
skill: 18-instagram-roteiro
→ aplicar estrutura: Gancho (3s) → Problema → Solução → CTA
```
