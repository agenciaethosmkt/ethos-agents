# Formatos Instagram — Especificações Técnicas

## Feed

| Parâmetro | Valor |
|---|---|
| Dimensões | 1080×1080 (quadrado) ou 1080×1350 (retrato) |
| Formato de arquivo | PNG ou JPG |
| Tamanho máximo | 2MB |
| Limite de caracteres legenda | 2.200 |
| Hashtags recomendadas | 5–10 (no comentário) |
| Publicação automática | ✅ via Make |

## Carrossel

| Parâmetro | Valor |
|---|---|
| Dimensões por slide | 1080×1080 |
| Nº de slides | 2–10 |
| Formato de arquivo | PNG por slide |
| Tamanho máximo por slide | 2MB |
| Limite de caracteres legenda | 2.200 |
| Slide 1 | Capa com gancho visual forte |
| Slide final | CTA + @perfil |
| Publicação automática | ✅ via Make |

## Stories

| Parâmetro | Valor |
|---|---|
| Dimensões | 1080×1920 (9:16) |
| Formato de arquivo | PNG por frame |
| Tamanho máximo | 2MB |
| Duração recomendada | 1–7 frames |
| Publicação automática | ❌ manual |
| Motivo | API do Instagram não suporta agendamento de Stories |

## Reels

| Parâmetro | Valor |
|---|---|
| Duração ideal | 30–60 segundos |
| Formato de arquivo | MP4 (produção externa) |
| Resolução | 1080×1920 (9:16) |
| Design automatizado | ❌ — apenas briefing visual |
| Publicação automática | ❌ manual |
| Motivo | Sem automação de vídeo configurada |

## E-mail Marketing

| Parâmetro | Valor |
|---|---|
| Assunto | Máximo 50 caracteres |
| Preview text | Máximo 90 caracteres |
| Corpo | 150–300 palavras |
| CTA | 1 único |
| Plataforma | Definida no briefing |
| Publicação automática | Depende da plataforma |

## YouTube (Video)

| Parâmetro | Valor |
|---|---|
| Formato | MP4 (produção externa) |
| Thumbnail | 1280×720 |
| Publicação automática | ❌ manual |
| Motivo | Sem automação configurada |
