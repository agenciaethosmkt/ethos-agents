# Anti-Padrões — Fábrica de Criativos

## Copy

- **Headline genérica:** "Conheça nosso produto", "Somos a melhor solução" → sempre específico
- **Começar com o nome da marca:** "A [Empresa] oferece..." → o usuário não se importa com a marca ainda
- **CTA vago:** "Clique aqui" sem contexto → "Garanta sua vaga" / "Calcule seu desconto" / etc.
- **Copiar ângulo já testado** sem mencionar histórico e por que vai funcionar desta vez
- **Variações idênticas:** apenas trocar uma palavra entre A e B não é um teste válido

## Design

- **Texto cobrindo > 20% da imagem** (Meta penaliza com menos alcance)
- **Muitos elementos competindo:** fundo complexo + logo + produto + texto + ícones = ruído
- **Fonte abaixo de 20px** em qualquer elemento de texto na imagem
- **Inconsistência com a marca do cliente:** cores aleatórias quando o cliente tem brand kit
- **Imagem sem relação com o copy:** visual que conta uma história diferente do texto
- **Dimensões erradas:** usar 1080×1080 quando o formato pedido é Stories (1080×1920)

## Processo

- **Pular o histórico:** nunca criar criativo sem verificar o que já foi testado para o cliente
- **Não consultar as Instruções de Design do Ogilvy** antes de criar o design
- **Entregar sem exportar** para revisão visual (o link do Canva não é suficiente — exportar PNG)
