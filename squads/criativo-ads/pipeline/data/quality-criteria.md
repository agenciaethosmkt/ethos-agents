# Critérios de Qualidade — Fábrica de Criativos

## Copy (Ogilvy)

| Critério | Padrão |
|---|---|
| Headline específica | Não pode ser genérica — deve conter dado, benefício ou pergunta específica |
| Hook nos primeiros 3 segundos | A primeira linha/imagem deve prender atenção imediatamente |
| CTA claro | Instrução de ação explícita — o usuário sabe exatamente o que fazer |
| Alinhamento com ângulo aprovado | Copy deve executar o ângulo selecionado, não inventar nova direção |
| Variações genuínas | A e B devem testar hipóteses diferentes (formato, gancho, oferta) |

## Design (Kizo/Designer)

| Critério | Padrão |
|---|---|
| Dimensões corretas | Deve seguir exatamente as especificações de `formatos-ads.md` |
| Texto na imagem | Máximo 20% da área (Meta) — preferir texto mínimo na imagem |
| Legibilidade | Nenhum texto abaixo de 20px na imagem |
| Hierarquia visual | 1 elemento principal dominante, não 5 elementos competindo |
| Congruência copy-visual | Imagem conta a mesma história que o copy — não contradiz |
| CTA visível | Botão ou instrução de ação clara |

## Revisão Final

| Critério | Verificação |
|---|---|
| Objetivo atendido | O criativo responde ao objetivo declarado no briefing? |
| Ângulo executado | Copy E design estão alinhados com o ângulo aprovado? |
| Pronto para subir | O criativo pode ser carregado no Gerenciador sem ajustes? |
