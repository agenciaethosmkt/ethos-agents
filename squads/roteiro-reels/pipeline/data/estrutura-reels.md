# Estrutura de Reels — Referência

## Dimensões e formato
- Proporção: 9:16 vertical
- Resolução: 1080×1920px
- Duração: 15s a 90s (ideal: 30–60s para conteúdo)

## Tabela de duração × cenas

| Duração alvo | Cenas | Ritmo | Melhor uso |
|---|---|---|---|
| 15s | 3–4 | Rápido, sem voz off longa | Gancho + 2 pontos + CTA |
| 30s | 5–7 | Médio | Lista curta, dica rápida |
| 45s | 8–10 | Narrativo | Tutorial 3 passos, storytelling curto |
| 60s | 10–14 | Completo | Tutorial detalhado, storytelling, antes/depois |

## Arcos narrativos disponíveis

| Arco | Objetivo | Estrutura |
|---|---|---|
| **Problema → Solução** | Autoridade, Conversão | Dor → Causa → Solução → CTA |
| **Lista** | Educação, Engajamento | Gancho → Item 1 → 2 → 3 → Conclusão → CTA |
| **Antes/Depois** | Prova Social, Conversão | Estado atual → Transformação → Resultado → CTA |
| **Tutorial passo a passo** | Autoridade | Objetivo → Passo 1 → 2 → 3 → Resultado → CTA |
| **Storytelling** | Engajamento, Relacionamento | Contexto → Conflito → Virada → Lição → CTA |
| **Trend/POV** | Alcance, Entretenimento | Contexto do trend → adaptação ao nicho → CTA leve |

## Formatos de gancho (primeiros 3s)

- **Contradição:** "Tudo que te ensinaram sobre X está errado"
- **Promessa concreta:** "Em 30 segundos você vai aprender X"
- **Curiosidade:** "O motivo pelo qual X não funciona para você"
- **Identidade:** "Se você é [perfil], presta atenção nisso"
- **Número/lista:** "3 erros que estão sabotando seu X"
- **Resultado:** "Como meu cliente saiu de X para Y em 30 dias"

## CTAs por objetivo

| Objetivo | CTAs recomendados |
|---|---|
| Engajamento | "Salva esse vídeo", "Manda nos comentários o seu X" |
| Conversão | "Manda 'quero' no direct", "Clica no link da bio para [benefício específico]" |
| Awareness | "Segue para não perder o próximo", "Compartilha com quem precisa ver isso" |
| Relacionamento | "Conta pra mim nos comentários", "Me manda uma mensagem" |

## Regras de texto na tela

- Máximo 5 palavras por frame para textos de destaque
- Legibilidade em mobile: fonte grande, alto contraste
- Texto na tela complementa a voz — nunca repete palavra por palavra
- Cena 1 (gancho): texto na tela obrigatório, aparece antes da voz terminar

## Sugestões de áudio

- **Trilha instrumental:** para tutoriais, educativos — não compete com a voz off
- **Trend musical:** para entretenimento, bastidores — aumenta alcance orgânico
- **Voz off sem música:** para storytelling e provas sociais — mais intimidade
- **Silêncio + legenda:** para públicos que assistem sem som (estimar 60% do alcance)
