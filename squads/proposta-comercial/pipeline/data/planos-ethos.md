# Planos Ethos — Escopo e Entregas

## Flow — Estruturação

**Para quem:** Clínicas pequenas (0–3 funcionários, R$10–20k/mês) e serviços técnicos locais que precisam organizar a aquisição de clientes.

**Valor referência:** ~R$2.000/mês (confirmar com Renato antes de inserir na proposta)

### Entregas incluídas

| Serviço | Frequência |
|---|---|
| Gestão de Instagram | 3–4 posts/semana (feed + stories) |
| Copywriting (copy + legendas) | Todos os posts |
| Design das peças | Todos os posts |
| Google Meu Negócio | Configuração + posts semanais |
| Tráfego pago (Meta Ads) | Campanha de geração de leads (verba à parte) |
| Relatório de performance | Mensal |
| Reunião de alinhamento | Mensal |
| Suporte via WhatsApp | Seg–Sex, horário comercial |

### Pilares do método (Flow)

1. **Presença** — Construir identidade digital consistente que gere confiança
2. **Atração** — Gerar demanda previsível via conteúdo + tráfego local
3. **Conversão** — Transformar interesse em agendamentos/contatos

## Growth — Expansão

**Para quem:** Clínicas com equipe (4–8 funcionários, R$20–50k/mês) que já fazem marketing mas de forma fragmentada e querem escalar com dados.

**Valor referência:** ~R$5.000/mês (confirmar com Renato antes de inserir na proposta)

### Entregas incluídas

| Serviço | Frequência |
|---|---|
| Gestão completa de Instagram | 5–7 posts/semana (feed + stories + reels) |
| Copywriting estratégico | Todos os conteúdos |
| Design premium | Todos os conteúdos |
| Tráfego pago Meta + Google | Campanhas ativas (verba à parte) |
| Gestão de públicos e criativos | Contínua |
| Dashboard de performance | Acesso em tempo real |
| Relatório executivo | Quinzenal |
| Reunião estratégica | Quinzenal |
| Suporte prioritário WhatsApp | Seg–Sex, horário comercial |

### Pilares do método (Growth)

1. **Dados** — Tomada de decisão orientada por métricas, não intuição
2. **Escala** — Aumentar investimento com previsibilidade de retorno (CAC, ROAS)
3. **Sistema** — Processo de aquisição que funciona sem depender de uma única ação
