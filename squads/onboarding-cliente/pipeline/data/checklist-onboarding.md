# Checklist de Onboarding por Plano

## Checklist geral (todos os planos)

### Validação de entrada
- [ ] Pagamento confirmado
- [ ] Contrato assinado
- [ ] Responsável do cliente identificado
- [ ] WhatsApp do responsável registrado
- [ ] CS interno designado

### Agendamento
- [ ] Mensagem de boas-vindas enviada (≤ 2 dias úteis após pagamento)
- [ ] Reunião agendada (≤ 2 dias úteis após contato)
- [ ] Link do Google Meet enviado ao cliente

### Reunião
- [ ] Briefing de reunião preparado pelo CS
- [ ] Reunião realizada
- [ ] Ata preenchida
- [ ] Objetivos e prioridades registrados
- [ ] Próximos passos alinhados com o cliente

### Pós-reunião
- [ ] Documento de próximos passos enviado ao cliente (WhatsApp)
- [ ] Passagem interna registrada no ClickUp
- [ ] Áreas notificadas (Tráfego, Conteúdo, CS)
- [ ] Task atualizada para "em revisão"

## Acessos necessários por área

### Tráfego Pago
- [ ] Acesso ao Business Manager (Meta)
- [ ] Acesso à conta Google Ads
- [ ] Acesso ao Google Analytics / GA4
- [ ] Acesso ao Pixel Meta instalado no site

### Conteúdo / Design
- [ ] Manual de marca (logo, cores, fontes)
- [ ] Acesso ao Instagram (perfil profissional)
- [ ] Acesso ao Canva (se tiver conta)
- [ ] Banco de imagens / fotos do cliente

### CS
- [ ] Acesso ao ClickUp (se for dar ao cliente)
- [ ] Grupo de WhatsApp criado (se plano incluir)

## Particularidades por plano

### Flow
- Automação Make já cria o projeto automaticamente
- Verificar se task foi criada corretamente antes de iniciar pipeline

### Growth / Business / Connect
- Verificar automação (pode não estar automatizado ainda)
- Criar manualmente se necessário

### Eletricistas
- Sem reunião de onboarding
- Formulário de briefing enviado via automação Make
- Verificar retorno do formulário antes de avançar
