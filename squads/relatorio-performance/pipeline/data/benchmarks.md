## Como usar

1. Identificar o setor do cliente
2. Comparar a métrica obtida com o range da tabela
3. Classificar: abaixo / dentro / acima da média
4. Contextualizar no relatório: "Seu CPL de R$18 está dentro da média para o setor de serviços locais (R$10–30)."
